# Air Quality Analysis

This project analyzes air pollution levels across Indian cities using Python.

## Steps
- Data Cleaning (handling NaN values)
- Line chart of AQI over time
- Bar chart of top polluted cities
- Pie chart for AQI category distribution
- Correlation heatmap
- Monthly AQI trend

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
